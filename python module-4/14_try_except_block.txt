14) How many except statements can a try-except block have? Name Some
built-in exception classes: 

1) SyntaxError: Raised when there is a syntax error in the code.
2)IndentationError: A subclass of SyntaxError, raised when there is an indentation error (e.g., inconsistent or incorrect indentation).
3)NameError: Raised when an identifier is not found in the local or global scope.
4) TypeError: Raised when an operation or function is applied to an object of inappropriate type.
5) ValueError:Raised when a function receives an argument of the correct data type but an inappropriate value.
